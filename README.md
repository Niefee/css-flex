# scss-flex

> scss mixin for Flex-layout module

## Installing & Usage

```bash
# install
$ npm install @nn-yy/scss-flex
```

```scss
// use with scss
@import '@nn-yy/scss-flex';

.container {
  @include flex( params );
  @include flex-wrap( params );
  // ...
}

.item {
  @include flex-self( params );
  // ...
}

```
